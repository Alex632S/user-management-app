ГРУППА I: Реактивность — сердце Vue 🌿

ref / shallowRefреактивность
const count = ref(0) → { value: 0 }
count.value++ → обновляет DOM
Базовая реактивная ссылка. Работает с примитивами и объектами. shallowRef — только поверхностно.
import { ref, shallowRef } from 'vue'
const age = ref(25)
const shallow = shallowRef({ deep: { nested: true } })
.value в JSглубокий трекинг
reactive / readonlyреактивность
reactive({ name: 'Vue', version: 3 }) → Proxy
Глубокий реактивный объект (Proxy). Не требует .value, но нельзя перезаписать целиком.
const state = reactive({ user: { name: 'Elena' } })
const frozen = readonly(state)
прямой доступreadonly защита
computedвычисляемое
computed(() => firstName.value + ' ' + lastName.value)
Кешируемое значение, пересчитывается только при изменении зависимостей.
const fullName = computed(() => user.first + ' ' + user.last)
fullName.value // геттер
ленивыйкеширование
toRef / toRefsреакт. утилиты
toRef(reactiveObj, 'key') → сохраняет связь
Создаёт ref из свойства реактивного объекта, сохраняя реактивность.
const state = reactive({ a:1, b:2 })
const { a } = toRefs(state)
деструктуризация без потери

ГРУППА II: Жизненные циклы компонента 🔄

onMounted / onBeforeMountмонтирование
onMounted — DOM готов, доступны ref элементов. onBeforeMount — перед первой отрисовкой.
onMounted(() => { console.log('Компонент в DOM') })
onBeforeMountonMounted
onUpdate / onBeforeUpdateобновление
Срабатывают при ререндере компонента (реактивные изменения).
onBeforeUpdate(() => console.log('скоро патч'))
onUpdated(() => console.log('DOM обновлён'))
onUnmounted / onBeforeUnmountразмонтирование
Очистка таймеров, слушателей, отписка от событий.
onUnmounted(() => clearInterval(intervalId))
onActivated / onDeactivatedKeepAlive
Для компонентов внутри <KeepAlive> — активация/деактивация кеша.
onActivated(() => fetchData())
onDeactivated(() => saveScroll())
onErrorCapturedошибки потомков
Перехват ошибок в дочерних компонентах, можно предотвратить всплытие.
onErrorCaptured((err, instance, info) => { logError(err); return false })
onRenderTracked/Triggereddevtools
Отладка: когда зависимость отслежена или вызвала ререндер (только dev).

ГРУППА III: Слежение за изменениями 👁️

watchисточник → callback
watch(source, (newVal, oldVal) => {...}, { deep, immediate })
Ленивый, следит за конкретными источниками, даёт доступ к старому значению.
watch(() => state.count, (val, old) => console.log(val, old))
watchEffectавтосбор зависимостей
Запускается немедленно, автоматически отслеживает все реактивные зависимости внутри.
watchEffect(() => { console.log(todoCount.value, user.name) })
cleanup функция
watchPostEffect / watchSyncEffectтайминги
watchPostEffect — после DOM-обновления. watchSyncEffect — синхронно до обновления.
watchPostEffect(() => measureDom())

ГРУППА IV: Компонентные инструменты (макросы) 🧩

defineProps / defineEmitsscript setup
Типизированные props и события, доступны без импорта в <script setup>.
const props = defineProps<{ title: string }>()
const emit = defineEmits<{ (e: 'close'): void }>()
defineExpose / defineOptionsкомпонентные
defineExpose — открыть методы/свойства для родительских ref. defineOptions — задать name, inheritAttrs.
defineExpose({ resetForm, validate })
useSlots / useAttrsконтекст
useSlots() — доступ к слотам, useAttrs() — все унаследованные атрибуты.
const slots = useSlots(); slots.default?.()
defineModel (Vue 3.4+)v-model упрощение
Двустороннее связывание без лишнего кода.
const modelValue = defineModel()
modelValue.value = 'new'

ГРУППА V: Dependency injection и продвинутые утилиты ⚙️

provide / injectDI
Передача данных через иерархию (независимо от уровня вложенности).
provide('theme', 'dark')
const theme = inject('theme', 'light')
effectScope / getCurrentScopeгруппировка эффектов
Объединяет эффекты (watch, computed) для массовой очистки.
const scope = effectScope(); scope.run(() => { watch(...) }); scope.stop()
customRef / triggerRefтонкий контроль
customRef — собственная логика трекинга (debounce, throttle). triggerRef — принудительное обновление shallowRef.
customRef((track, trigger) => { get() { track(); return value }, set(v) { value=v; trigger() }})
markRaw / toRawисключения реактивности
markRaw — запретить реактивность объекта. toRaw — получить оригинал из прокси.
const obj = markRaw({ noReactive: true })
const raw = toRaw(reactiveObj)

ГРУППА VI: Встроенные компоненты + глобальные инструменты 🧩

<Teleport>порталы
Рендер в другой DOM-узел (модалки, тосты).
<Teleport to="body"><Modal /></Teleport>
<Transition> / KeepAliveанимация + кеш
Transition — анимация появления/исчезновения. KeepAlive — кеширование компонентов.
nextTick / versionутилиты рантайма
nextTick — дождаться обновления DOM. version — текущая версия Vue.
await nextTick(); console.log('DOM обновлён')
app.use / app.componentглобальная регистрация
Плагины, глобальные компоненты и директивы.

💻 Полный пример компонента (Vue 3 + script setup) 

<script setup>
import { ref, reactive, computed, onMounted, watchEffect, defineProps, defineEmits } from 'vue'

const props = defineProps({ initialCount: Number })
const emit = defineEmits(['update'])

const count = ref(props.initialCount ?? 0)
const double = computed(() => count.value * 2)

const state = reactive({ history: [] })

function increment() {
  count.value++
  state.history.push(`+1 → ${count.value}`)
  emit('update', count.value)
}

watchEffect(() => {
  console.log(`Счётчик стал: ${count.value}`)
})

onMounted(() => {
  console.log('Компонент смонтирован')
})
</script>

<template>
  <div>
    <button @click="increment">{{ count }} (x2 = {{ double }})</button>
    <ul><li v-for="log in state.history" :key="log">{{ log }}</li></ul>
  </div>
</template>
